Let's see, what do I have to say. Well, let's start with my site mechanics.
  • Navigation: You can navigate thru my site in a couple of ways.
    1. Below the header of each page (in this case "Welcome to ... The official... Skemp") there is a series of links to every single page.
    2. At the bottom of the page, directly above the notepad gif, there is a drop box that can be used. Simply select the topic, and press the 'go' button.
  • Pages: All of my pages have;
    1. A header, short description of page.
    2. Links to all the other pages (noted above).
    3. A link to email me (jrs_webdesign@yahoo.com) if there are any problems with the site or if you want me to make you a site (note however that I can't do a lot of requests).
    4. A counter (actually a cookie) that tells you how many times you have been to the site. Note: if you don't come here for a week, the cookie should expire (be deleted) although it might before then.
Well, that's all for now. Go back up to the top of the page, and click on a link, or continue to the bottom and use the drop-down menu. Thanks and enjoy!
